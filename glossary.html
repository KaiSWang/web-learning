所有縮寫術語
一次查清楚

Web 領域充滿縮寫。這份對照表收集了本系列出現過的所有縮寫，按主題分類。 讀任何章節時看到不熟的縮寫，回來這頁查一下就知道是什麼。

▸ 網路協定 · Protocols
HTTP
HyperText Transfer Protocol
網頁通訊的基礎協定，請求-回應模型
HTTPS
HTTP Secure
HTTP + TLS 加密版本，網址前面有 🔒
TCP
Transmission Control Protocol
確保資料可靠送達的傳輸協定，三次握手
UDP
User Datagram Protocol
較快但不可靠的傳輸協定，遊戲、串流用
TLS
Transport Layer Security
傳輸層加密協定，舊名 SSL
SSL
Secure Sockets Layer
TLS 的舊稱，現在多指 SSL/TLS 憑證
DNS
Domain Name System
把網域名稱（google.com）翻譯成 IP 的系統
IP
Internet Protocol
網路定位協定，每台機器有獨特 IP 位址
ISP
Internet Service Provider
網路服務供應商，如中華電信、Hinet
TTL
Time To Live
DNS / 快取的存活時間
CDN
Content Delivery Network
分散全球的快取網路（Cloudflare 等）
PoP
Point of Presence
CDN 的邊緣節點
▸ 前端技術 · Frontend
HTML
HyperText Markup Language
網頁結構標記語言
CSS
Cascading Style Sheets
樣式語言，控制網頁外觀
JS
JavaScript
瀏覽器內的程式語言，負責互動
DOM
Document Object Model
HTML 解析後的物件樹，JS 操作的對象
CSSOM
CSS Object Model
CSS 解析後的物件樹
SPA
Single-Page Application
單頁應用，靠 JS 切換內容不重整
MPA
Multi-Page Application
多頁式應用，每次點連結重新載入
SSR
Server-Side Rendering
伺服器端渲染，每次請求即時產生 HTML
SSG
Static Site Generation
建置時就產生好靜態 HTML（最快）
CSR
Client-Side Rendering
瀏覽器端渲染，傳統 SPA 模式
SEO
Search Engine Optimization
搜尋引擎優化，讓 Google 容易讀懂頁面
PWA
Progressive Web App
可離線運作、可裝到手機主畫面的網頁
RWD
Responsive Web Design
響應式設計，一份程式碼適應所有螢幕
a11y
Accessibility
無障礙，讓有障礙的使用者也能用
▸ API 與資料 · APIs & Data
API
Application Programming Interface
應用程式介面，後端開放給前端呼叫的入口
REST
Representational State Transfer
主流的 API 設計風格，用 HTTP 動詞操作資源
GraphQL
Graph Query Language
Facebook 開發的查詢語言，前端決定要拿什麼欄位
JSON
JavaScript Object Notation
資料交換格式，現代 API 主流
XML
eXtensible Markup Language
較舊的資料格式，仍用於某些企業系統
AJAX
Asynchronous JavaScript and XML
非同步請求技術，現代多用 fetch 取代
SQL
Structured Query Language
關聯式資料庫的查詢語言
NoSQL
Not Only SQL
非關聯式資料庫的統稱
ORM
Object-Relational Mapping
用程式物件操作資料庫的工具
CRUD
Create, Read, Update, Delete
資料操作的四個基本動作
▸ 認證與安全 · Auth & Security
OAuth
Open Authorization
授權協定（不是認證），「用 Google 登入」用的就是這個
OIDC
OpenID Connect
在 OAuth 上加上身份識別的標準
SSO
Single Sign-On
單一登入，登入一次多服務通用
JWT
JSON Web Token
把使用者資訊編碼進 token 的格式
2FA
Two-Factor Authentication
雙因素驗證
MFA
Multi-Factor Authentication
多因素驗證（2FA 是其中一種）
OTP
One-Time Password
一次性密碼（簡訊驗證碼）
TOTP
Time-based One-Time Password
時間型一次性密碼，Google Authenticator 用的
XSS
Cross-Site Scripting
跨站腳本攻擊，注入惡意 JS 到頁面
CSRF
Cross-Site Request Forgery
跨站請求偽造，借用使用者身份送請求
CORS
Cross-Origin Resource Sharing
跨來源資源共享，瀏覽器的安全機制
CSP
Content Security Policy
內容安全政策，限制可載入的資源來源
DDoS
Distributed Denial of Service
分散式阻斷服務攻擊，用流量灌爆網站
WAF
Web Application Firewall
網頁應用程式防火牆，過濾攻擊請求
SBOM
Software Bill of Materials
軟體物料清單，記錄所有依賴
▸ 效能指標 · Performance Metrics
TTFB
Time To First Byte
收到第一個 byte 的時間，反映伺服器速度
FCP
First Contentful Paint
第一個內容元素出現的時間
LCP
Largest Contentful Paint
最大內容元素出現的時間（Core Web Vital）
CLS
Cumulative Layout Shift
累計版面位移，元素亂跳的程度
INP
Interaction to Next Paint
互動到下一次繪製的時間（取代 FID）
FID
First Input Delay
首次輸入延遲（已被 INP 取代）
TTI
Time To Interactive
頁面變得可互動的時間
RUM
Real User Monitoring
真實使用者監控，蒐集生產環境數據
▸ 後端與基礎設施 · Backend & Infrastructure
LB
Load Balancer
負載均衡器，分配流量到多台伺服器
VM
Virtual Machine
虛擬機，雲端伺服器的單位
k8s
Kubernetes
容器調度系統，管理數百台機器
CI
Continuous Integration
持續整合，自動測試新程式碼
CD
Continuous Deployment
持續部署，自動上線
SaaS
Software as a Service
軟體即服務，如 Notion、Slack
PaaS
Platform as a Service
平台即服務，如 Vercel、Heroku
IaaS
Infrastructure as a Service
基礎設施即服務，如 AWS EC2
SLA
Service Level Agreement
服務品質協議（如 99.9% 可用性）
▸ 其他常見縮寫 · Misc
SSE
Server-Sent Events
伺服器推送事件，單向即時通訊
UX
User Experience
使用者體驗
UI
User Interface
使用者介面
IA
Information Architecture
資訊架構，頁面與內容如何分層
PR
Pull Request
合併請求，Git 協作的核心動作
MVP
Minimum Viable Product
最小可行產品
A11y / I18n
Accessibility / Internationalization
數字代表中間字數（a-11個字母-y）
P2P
Peer-to-Peer
點對點，不經過伺服器
